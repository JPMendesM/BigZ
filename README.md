# 🌊 BigZ – Surfando em Hábitos Saudáveis

Sistema web desenvolvido para incentivar a adoção de hábitos saudáveis e sustentáveis por meio de registro de atividades, acompanhamento de progresso, pontuação e interação entre usuários.

O projeto foi desenvolvido utilizando **Elixir**, **Phoenix LiveView** e **PostgreSQL**, com foco em programação funcional, autenticação segura e experiência interativa para o usuário.

# 🎯 Objetivo

O BigZ tem como objetivo auxiliar usuários na construção de hábitos saudáveis por meio de uma plataforma simples e interativa, permitindo o acompanhamento de metas pessoais e incentivando a constância através de pontuações e futuras funcionalidades de gamificação.

# ✅ Funcionalidades Implementadas (Módulo A)

## Cadastro de Usuários

* Cadastro com nome
* Cadastro com e-mail
* Cadastro com senha
* Validação de dados
* Senha armazenada de forma criptografada

## Autenticação

* Login por e-mail e senha
* Logout seguro
* Sessão persistente
* Suporte a Magic Link

## Perfil do Usuário

* Visualização do perfil
* Edição do nome
* Edição da bio
* Exibição da pontuação acumulada

## Banco de Dados

* Integração com PostgreSQL
* Hospedagem utilizando Supabase
* Persistência completa dos dados dos usuários
